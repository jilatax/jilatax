# Jilatax documentation

The independently deployed documentation application for Jilatax.

- Public URL: `https://doc.jilatax.dev`
- Framework: Astro + Starlight
- Documentation content: `src/content/docs/`

## Commands

```bash
bun install
bun run dev
bun run build
```

The marketing website lives separately at `apps/website` and is deployed to
`https://jilatax.dev`.
