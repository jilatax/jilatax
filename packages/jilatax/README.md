# jilatax

> A minimal TypeScript library package with ESM, CommonJS, and declaration outputs.

Requires Node.js 22.18 or newer.

## Install

```bash
npm i jilatax
```

## Usage

```ts
import { hello } from "jilatax";

hello("world");
```

## Development

```bash
bun install
bun run check
```
